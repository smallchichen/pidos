## eclipse xml字體大小
* https://usit-yuan.blogspot.com/2013/04/eclipse.html
<pre>
修改編輯區字體大小
    1. Window > Preference
    2. 右側General > Appearance > Colors and Fonts
    3. 將 Basic 展開，看到「Text Font」和「Text Editor Block Selection Font」
    4. 按下 Edit ，隨自己喜好調整即可。
Text Font就可以改變字體大小    
</pre>
